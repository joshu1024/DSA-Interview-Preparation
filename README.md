## Problem Template

For every problem solved:

* Problem Name
* Pattern
* Difficulty
* Approach
* Time Complexity
* Space Complexity
* Optimized JavaScript Solution

Example:

```javascript
/*
Problem: Two Sum
Pattern: Hash Map
Difficulty: Easy

Brute Force:
Check every pair.
Time: O(n²)

Optimized:
Use a hash map to store previously seen numbers.

Time: O(n)
Space: O(n)
*/
```
